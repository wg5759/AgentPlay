# 贡献者公约行为准则

## 我们的承诺

作为成员、贡献者和领导者，我们承诺让社区的每个人都能获得无骚扰的参与体验，无论其年龄、体型、可见或不可见的残障、族裔、性别特征、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、个人形象、种族、宗教或性认同与取向如何。

我们承诺以有助于建设开放、友善、多元、包容和健康社区的方式行动和互动。

## 我们的标准

有助于创造积极环境的行为包括：

- 对他人表现出同理心和善意
- 尊重不同的意见、观点和经验
- 给予并优雅地接受建设性反馈
- 为自己的错误承担责任并向受影响的人道歉
- 关注整个社区的最大利益，而不仅仅是个人利益

不可接受的行为包括：

- 使用性化的语言或图像，以及任何形式的性关注或挑逗
- 挑衅、侮辱或贬损性评论，以及人身或政治攻击
- 公开或私下的骚扰
- 未经明确许可发布他人的私人信息（如真实地址或电子邮箱）
- 其他在专业场合可被合理认定为不当的行为

## 执行责任

社区领导者负责澄清和执行本行为准则，并针对其认为不当、威胁、冒犯或有害的任何行为采取适当、公正的纠正措施，包括删除、编辑或拒绝不符合本准则的评论、提交、代码、wiki 编辑、问题和其他贡献，并在适当时说明 moderation 决定的理由。

## 适用范围

本行为准则适用于所有社区空间，也适用于个人在公共场所正式代表社区的情形（如使用官方电子邮箱、通过官方社交媒体账号发布内容，或在线上/线下活动中担任指定代表）。

## 执行

如有辱骂、骚扰或其他不可接受的行为，请通过 GitHub Issues（<https://github.com/wg5759/AgentPlay/issues>）向社区领导者报告。所有投诉都将得到及时、公正的审查和调查，并尊重报告者的隐私与安全。

## 执行指南

社区领导者将遵循以下社区影响指南来决定违规行为的后果：

### 1. 纠正

**社区影响**：使用不当语言或其他被认为不专业或不受欢迎的行为。
**后果**：私下书面警告，说明违规性质并要求澄清；可要求公开道歉。

### 2. 警告

**社区影响**：单次事件或一系列行为造成的违规。
**后果**：书面警告，并在规定时间内禁止与相关人员互动（包括 unsolicited 互动）；违反者可能被临时限制参与社区。

### 3. 临时封禁

**社区影响**：严重或持续的违规行为。
**后果**：在规定时间内临时禁止与社区进行任何形式的互动或公开交流。

### 4. 永久封禁

**社区影响**：表现出持续的违规模式、骚扰个人或群体、或攻击/贬低个人或群体。
**后果**：永久禁止在社区内进行任何形式的公开互动。
